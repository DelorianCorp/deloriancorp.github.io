# Comando ```inicio```

El comando ```inicio``` marca, como lo dice su nombre, el inicio de una función en el programa. Como es un comando de apertura, en vez de usarse con un punto y coma (;) al final, se usa con dos puntos (:).

### Sintaxis:
```sh
inicio:
```

### Ejemplo:
```sh
# Función vacía
inicio:
fin;
```
